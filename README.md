# FactuLink

FactuLink es la interfaz frontend de una aplicación web diseñada para gestionar inventario, cotizaciones, compras y ventas de manera eficiente. Facilita la administración de productos y operaciones comerciales dentro de una empresa.

## Funcionalidades

- Gestión de inventario de productos
- Creación y administración de cotizaciones
- Control de compras y ventas
- Administración de clientes y proveedores
- Reportes y seguimiento de operaciones

## Estado del proyecto

Este proyecto está en desarrollo activo y será mejorado continuamente con nuevas funcionalidades y optimizaciones.

## Próximas integraciones

- Integración con SUNAT para facturación electrónica

## Tecnologías

- [Angular CLI](https://github.com/angular/angular-cli) v20.3.3
- [Core UI](https://coreui.io/angular/docs/getting-started/introduction/) v5
- TypeScript, SCSS, Bootstrap

## Instalación

Clona el repositorio:

```bash
git clone https://github.com/Hearly98/factulink-front.git
cd factulink-front
```

Instala las dependencias:

```bash
npm install
```

Ejecuta el servidor de desarrollo:

```bash
ng serve
```

Abre tu navegador en `http://localhost:4200`

## Nota importante

Este repositorio contiene únicamente el frontend de la aplicación. El backend es un proyecto separado que no estáincluido en este repositorio.
